# Nexus Plan — ملخص المشروع (ما تم تنفيذه)

هذا الملف يوثّق **ما تم بناؤه فعلياً** في مشروع **منصة التخطيط المالي وتوقعات المبيعات** (Enterprise Sales Forecasting & Revenue Planning). يمكنك تنزيله من مجلد المشروع أو نسخه من المحرر.

---

## نظرة عامة

تم إنشاء تطبيق ويب احترافي (أساس إنتاجي) يجمع:

- لوحة تحكم تنفيذية مع مؤشرات مالية ومخططات
- محرك حسابات للتوقعات وعائد الاستثمار (ROI) والهدف من المبيعات
- محاكاة سيناريوهات متعددة
- إدارة شركات وأهداف وتدفقات إيرادات (بيانات تجريبية + تخزين محلي)
- خط أنابيب مبيعات مع تعديلات مباشرة
- شبكة توقعات شبيهة بجداول Excel
- مساعد ذكاء أعمال (قواعد جاهزة + API جاهز لاستبداله بنموذج لغوي لاحقاً)
- مخطط قاعدة بيانات PostgreSQL لـ **Supabase** مع RLS وصلاحيات

التطبيق يعمل **بدون Supabase** (وضع تجريبي كامل)، ويمكن تشغيله مع Supabase عند إضافة مفاتيح البيئة.

## محرك الحسابات (باختصار)

- **ROI** ≈ صافي الربح ÷ التكاليف الثابتة الشهرية  
- **هدف المبيعات (إيراد)** ≈ التكاليف الثابتة ÷ (هامش المساهمة − هدف صافي الربح كنسبة)، مع حماية من القسمة على صفر  
- **السيناريو:** يعدّل النمو، التحويل، التكاليف الثابتة، مزيج الإيرادات، وزن الأنابيب، وهدف NP ثم يعيد تشغيل المحرك  

التفاصيل في الملف: `src/lib/calculations/engine.ts`

## قاعدة البيانات (Supabase / PostgreSQL)

الملف: `supabase/migrations/001_initial_schema.sql`

يتضمن جداول مثل: المنظمات، الأعضاء، الشركات، المحافظ، وحدات الأعمال، تدفقات الإيراد، شرائح حجم الصفقة، السيناريوهات، التوقعات الشهرية، الفرص، لقطات KPI، التقارير، التعليقات، الموافقات، الأنشطة، **سجل التدقيق**، وأنواع أدوار (`admin`, `executive`, …).

مفعّل **Row Level Security (RLS)** مع سياسات مرتبطة بالعضوية في المنظمة.

## ربط Supabase (اختياري)

1. أنشئ مشروعاً على [Supabase](https://supabase.com).  
2. نفّذ محتوى `supabase/migrations/001_initial_schema.sql` في محرر SQL.  
3. انسخ `.env.example` إلى `.env.local` واملأ:  
   - `NEXT_PUBLIC_SUPABASE_URL`  
   - `NEXT_PUBLIC_SUPABASE_ANON_KEY`  
4. لإجبار تسجيل الدخول على كل المسارات: `NEXT_PUBLIC_REQUIRE_AUTH=true`  
5. لاستكمال تسجيل الدخول بالبريد/كلمة المرور مع PKCE، تأكد من إعداد **Redirect URL** في لوحة Supabase ليشمل مسار التطبيق: `/auth/callback`
